# nmf-common-cause

Code for the paper:

**Nonnegative matrix factorization and the principle of the common cause**  
E. Khalafyan, A. Allahverdyan and A. Hovhannisyan  
  

## Overview

This repository provides the code accompanying the paper, along with scripts for reproducing the main results presented in the paper.

## Requirements
Python >= 3.9

See requirements.txt for the full list of dependencies.
